# Lost & Found Item Management System

The **Lost & Found Item Management System** is a simple full-stack web application designed to help users report, view, and manage lost or found items. Users can submit item details such as name, description, location, and contact information, which are stored in a cloud database and displayed in a clean, organized interface.

This project uses:
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js + Express  
- **Database:** MongoDB Atlas  
- **Hosting:** GitHub Pages (Frontend) + Local/Cloud Backend
